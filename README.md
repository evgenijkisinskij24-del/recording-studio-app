# Приложение студии звукозаписи
Учебный проект в рамках лабораторной работы. 

## Описание
Desktop-приложение для взаимодействия с продюсерами, выпуска материалов и финансового учета. 

## Технологический стек
**Язык:** Python 
**Библиотека:** PyQt6 
**СУБД:** SQLite 
**Среда:** Visual Studio 2022 

## Структура проекта
`Forms/` — интерфейсы окон 
`Classes/` — программная логика 
`Data/` — база данных
