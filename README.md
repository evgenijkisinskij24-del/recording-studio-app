Студия звукозаписи: Учет материалов и финансовВерсия 0.1 (Учебный проект)   Десктопное приложение для управления процессами в студии звукозаписи: от поиска исполнителей и заключения договоров до выпуска альбомов и формирования финансовой отчетности.  📋 Функциональные возможностиСистема реализует следующие требования (FR):FR-01: Авторизация — вход по логину и паролю с разграничением прав доступа.  FR-02: Справочник «Клиенты» — ведение базы исполнителей (ФИО, телефон, райдер) с возможностью поиска.  FR-03: Справочник «Договоры» — создание договоров для конкретных клиентов, учет сумм и сроков.  FR-04: Регистрация обращений — оформление заявок на запись альбомов (SP/LP/Ep) с привязкой к действующим договорам.  FR-07: Учет оплат — отслеживание статусов оплаты договоров и счетов.  FR-08: Отчеты — формирование выгрузок по задолженностям, истекающим договорам и обращениям за период.  🛠 Технический стекЯзык программирования: Python.  Интерфейс: PyQt6 (библиотека PyQt).  База данных: SQLite (параметризованные SQL-запросы для защиты от инъекций).  Среда разработки: Visual Studio 2022 / PyCharm.  📂 Структура проектаКод структурирован согласно проектным ограничениям:  /Data — менеджер базы данных и файл БД (studio.db).  /Forms — все графические окна приложения (авторизация, клиенты, договоры и т.д.).  /Classes — программная логика и описания сущностей.  main.py — точка входа в приложение.🚀 Запуск и установкаКлонирование репозитория:Bashgit clone https://github.com/evgenijkisinskij24-del/recording-studio-app.git
