# Приложение студии звукозаписи

Учебный проект в рамках лабораторной работы.

## Описание

Desktop-приложение для взаимодействия с продюсерами, выпуска материалов и финансового учета.

## Технологический стек

* **Язык:** Python
* **Библиотека:** PyQt6
* **СУБД:** SQLite
* **Среда:** Visual Studio 2022 / PyCharm

## Структура проекта

* `Forms/` — интерфейсы окон (авторизация, клиенты, договоры, заявки, отчеты)
* `Data/` — менеджер базы данных и файл studio.db
* `main.py` — точка входа в приложение

## Функциональные возможности

* **FR-01: Авторизация** — вход по логину и паролю с разграничением прав доступа.
* **FR-02: Справочник «Клиенты»** — ведение базы исполнителей (ФИО, телефон, райдер) с возможностью поиска.
* **FR-03: Справочник «Договоры»** — создание договоров для конкретных клиентов, учет сумм и сроков.
* **FR-04: Регистрация обращений** — оформление заявок на запись альбомов (SP/LP/Ep) с привязкой к договорам.
* **FR-05: Учет оплат** — отслеживание статусов оплаты договоров и счетов.
* **FR-06: Отчеты** — формирование выгрузок по задолженностям и обращениям за период.

## Запуск и установка
